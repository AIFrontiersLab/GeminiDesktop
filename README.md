# Gemini Electron App

Simple macOS Electron desktop app that opens Google Gemini in a native window.

## Run

```bash
npm install
npm start
```

## Build macOS app

```bash
npm run build:mac
```

Build artifacts are generated in `dist/`:
- `Gemini Desktop-<version>-mac.zip`
- unpacked app bundle in `dist/mac-arm64/Gemini Desktop.app`
